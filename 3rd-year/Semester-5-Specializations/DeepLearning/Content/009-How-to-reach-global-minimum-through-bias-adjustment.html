We are going to simplify the scenario even further to make the concept of reaching a global minimum through bias adjustment incredibly clear.

We'll imagine the simplest possible "neural network": a single neuron with no inputs (or its input is always zero) and only a bias term.

Its job is to simply learn a constant output value.

The Super Simple Problem: Learning a Constant Value¶

Imagine we want our neuron to consistently output 0.75. Since there are no real inputs to consider (or we can assume $X=0$ and $W=0$), the neuron's output will solely depend on its bias, $c$, after passing through an activation function (let's use Sigmoid again).

  • Our Neuron's Formula: $Output = \\text{sigmoid}(c)$
  • Target Output: $y\_{target} = 0.75$
  • Loss Function: Mean Squared Error (MSE) = $(Output - y\_{target})^2$

Our goal is to find the value of $c$ that makes $\\text{sigmoid}(c)$ as close to $0.75$ as possible, thereby minimizing the MSE loss. In this 1-dimensional problem (only $c$ to adjust), the loss landscape will be a simple 2D curve with a single, obvious global minimum.

Explanation: The "Single Hill" Analogy¶

  1. The Goal: Our neuron's output is sigmoid(bias). We want this to be 0.75. To achieve 0.75 from a sigmoid function, the input to the sigmoid (which is just our bias in this case) needs to be approximately 1.0986. This is our global minimum target for the bias.

  2. The "Hill" (Loss Landscape):

    • The gray line in the plot represents our loss landscape. The horizontal axis is the Bias Value, and the vertical axis is the MSE Loss.
    • You can see it forms a clear bowl shape. This specific shape indicates a convex problem, meaning there's only one lowest point the global minimum. There are no other "dips" or "valleys" to get stuck in (no local minima or saddle points).
    • The blue star (*) marks this exact global minimum at bias 1.0986 where the loss is virtually zero.

  3. The "Blindfolded Man" (Gradient Descent):

    • We start our current_bias at 0.0 (the green triangle ^). The loss at this point is quite high (since sigmoid(0) = 0.5, and our target is 0.75).
    • In each epoch:
      • Feel the Slope (Calculate Gradient): We calculate gradient_bias. This tells us which way is "downhill" on the loss landscape for our current bias.
        • If our bias is too low (e.g., 0.0), our output 0.5 is less than 0.75. The gradient will be positive, indicating that increasing the bias will move us towards the minimum.
        • If our bias were too high (e.g., 3.0), our output sigmoid(3.0) 0.95 is greater than 0.75. The gradient would be negative, indicating that decreasing the bias will move us towards the minimum.
      • Take a Step (Update Bias): current_bias = current_bias - learning_rate * gradient_bias. We subtract the gradient to move in the downhill direction.
    • The red circles (o) trace the optimization path of our bias value across the loss landscape.

  4. Reaching the Global Minimum:

    • As the training progresses, you'll observe the Current Bias gradually increase and get closer to 1.0986.
    • Simultaneously, the Loss value will consistently decrease, getting closer and closer to zero.
    • The red path on the plot clearly shows the bias value moving directly towards the blue star, which is the global minimum. It's not getting stuck, because there's only one clear "bottom of the bowl."

This example beautifully illustrates how gradient descent iteratively adjusts a parameter (our bias here) by "feeling the slope" (calculating the gradient) and moving downhill, directly leading it to the lowest possible error, which is the global minimum for this simple, convex problem.

import numpy as np
import matplotlib.pyplot as plt

# --- 1. Activation Function ---
def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def sigmoid_derivative(x):
    s = sigmoid(x)
    return s * (1 - s)

# --- 2. Loss Function ---
def mse_loss(y_pred, y_true):
    return (y_pred - y_true)**2 # Squaring the difference (no mean needed for single sample)

# --- 3. Training Process for Only Bias ---
def train_bias_only(y_target, initial_bias, learning_rate, epochs):
    """
    Trains a neuron to learn a constant output by adjusting only its bias.
    """
    current_bias = initial_bias
    loss_history = []
    bias_history = []

    print(f"--- Training Bias Only (Target: {y_target}, LR: {learning_rate}) ---")
    print(f"Initial Bias: {current_bias:.4f}")

    for epoch in range(epochs):
        # --- Forward Pass ---
        output = sigmoid(current_bias)
        loss = mse_loss(output, y_target)

        # Store history
        loss_history.append(loss)
        bias_history.append(current_bias)

        # --- Backward Pass (Calculate Gradient for Bias) ---
        # dLoss/dOutput = 2 * (Output - y_target)
        d_output = 2 * (output - y_target)

        # dOutput/dBias = derivative of sigmoid(current_bias)
        d_bias_activation = sigmoid_derivative(current_bias)

        # Gradient of Loss w.r.t. Bias (dLoss/dBias) = dLoss/dOutput * dOutput/dBias
        gradient_bias = d_output * d_bias_activation

        # --- Update Bias ---
        current_bias = current_bias - learning_rate * gradient_bias

        if epoch % (epochs // 5) == 0 or epoch == epochs - 1:
            print(f"Epoch {epoch:4d} | Loss: {loss:.6f} | Current Bias: {current_bias:.4f} | Output: {output:.4f}")
            if epoch == 0:
                print(f"    Initial Gradient (dLoss/dBias): {gradient_bias:.4f}")

    return current_bias, loss_history, bias_history

# --- 5. Visualization: The Loss Landscape ---

# Calculate a range of bias values to plot the loss curve
bias_values = np.linspace(-2, 4, 100)
loss_landscape = [mse_loss(sigmoid(b), y_target) for b in bias_values]

# Calculate the optimal bias for y_target = 0.75
# sigmoid(b_optimal) = 0.75  =>  1 / (1 + exp(-b_optimal)) = 0.75
# => 1 + exp(-b_optimal) = 1 / 0.75 = 4/3
# => exp(-b_optimal) = 4/3 - 1 = 1/3
# => -b_optimal = ln(1/3) = -ln(3)
# => b_optimal = ln(3) approx 1.0986
optimal_bias = np.log(y_target / (1 - y_target))
optimal_loss = mse_loss(sigmoid(optimal_bias), y_target)

plt.figure(figsize=(10, 6))
plt.plot(bias_values, loss_landscape, label='Loss Landscape (MSE vs. Bias)', color='gray')
plt.scatter(bias_hist, loss_hist, color='red', marker='o', s=50, label='Optimization Path', zorder=5) # Plot trajectory
plt.scatter(optimal_bias, optimal_loss, color='blue', marker='*', s=200, label='Global Minimum', zorder=6) # Mark global minimum
plt.scatter(initial_bias, loss_hist[0], color='green', marker='^', s=100, label='Starting Point', zorder=6) # Mark start

plt.title('Loss Landscape for Single Bias Neuron: Reaching Global Minimum')
plt.xlabel('Bias Value')
plt.ylabel('Mean Squared Error Loss')
plt.grid(True)
plt.legend()
plt.ylim(bottom=0) # Ensure loss starts at 0 on y-axis
plt.show()
