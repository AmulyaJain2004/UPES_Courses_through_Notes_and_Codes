How weights and bias are adjusted?¶

This Python code explicitly shows how weights and biases are adjusted during the learning process, aiming to reach a global minimum.

Instead of a full neural network, we'll model a single "neuron" (or a very simple linear regression model) that tries to predict a value based on one input feature. This makes the concept of weights (just one here) and biases (just one here) and their adjustments very clear in a 2D plot.

The Problem: Simple Linear Regression¶

Let's assume we have some data points that roughly follow a linear relationship, like: $y = m * x + c$

Where:

  • $y$ is the output (target)
  • $x$ is the input
  • $m$ is the slope (our "weight")
  • $c$ is the y-intercept (our "bias")

Our "neuron" will try to learn $m$ and $c$ to best fit this data.

Goal: Find the values of $m$ and $c$ that minimize the Mean Squared Error (MSE) between our predictions and the actual data. This minimum will be our "global minimum" in this simple convex problem.

Explanation¶

  1. Generate Synthetic Data:

    • We create X (our input feature) and y_true (our target output).
    • y_true = true_m * X + true_c + np.random.randn(...) simulates data from a line y = 2x + 5 with some random noise, mimicking real-world data.

  2. mse_loss(y_pred, y_true):

    • This is our Loss Function. It quantifies how "bad" our current prediction y_pred is compared to the y_true.
    • Goal of Learning: We want to minimize this mse_loss. The minimum value of this function corresponds to the best possible m and c for our data. In this simple linear case, this minimum is guaranteed to be a global minimum (the lowest point in the entire landscape).

  3. linear_model(X, m, c):

    • This is our very simple "neuron" or model. It takes an input X, multiplies it by a weight (m), and adds a bias (c) to make a prediction.

  4. train_linear_model(...):

    • Initialization: We start with arbitrary initial_m and initial_c (our initial "guesses" for the weight and bias).
    • Epoch Loop: The training happens over a number of epochs (iterations).
    • Forward Pass:
      • y_pred = linear_model(X, m, c): We use our current m and c to make predictions.
      • loss = mse_loss(y_pred, y_true): We calculate how bad these predictions are.
    • Backward Pass (Gradient Calculation):
      • error_term = (y_pred - y_true): This is the fundamental error signal. If y_pred is too high, this is positive; if too low, it's negative.
      • gradient_m = np.mean(error_term * X): This is the derivative of the MSE loss with respect to m (our weight).
        • Intuitively:
          • If error_term (how much our prediction is off) is positive (we predicted too high) and X is positive, then gradient_m will be positive. This means increasing m made the error worse, so we need to decrease m.
          • If error_term is positive and X is negative, gradient_m will be negative. This means increasing m made the error better (reduced the predicted value), so we need to increase m.
          • This is exactly why we subtract the gradient.
      • gradient_c = np.mean(error_term * 1): This is the derivative of the MSE loss with respect to c (our bias).
        • Intuitively: If error_term is positive (we predicted too high), gradient_c will be positive. We need to decrease c to bring the prediction down.
    • Update Weights and Bias:
      • m = m - learning_rate * gradient_m
      • c = c - learning_rate * gradient_c
      • This is the core gradient descent step. We adjust m and c by moving them in the direction that decreases the loss, scaled by the learning_rate.

  5. Run the Training:

    • We set initial values and a learning_rate. You'll see the m and c values gradually adjust with each epoch, and the Loss steadily decreases.

  6. Visualization of Learning:

    • The first plot shows the true underlying line, our bad initial guess, and how our learned line progressively moves closer to the true line.
    • The second plot shows the Loss decreasing over epochs, visually demonstrating that our model is learning and getting better.

  7. Visualization of Weight and Bias Trajectory (Loss Landscape):

    • This is the most direct visualization of "reaching the global minimum."
    • We create a 3D plot where the X-axis is m, the Y-axis is c, and the Z-axis is the Loss. This forms a bowl-shaped surface (a convex loss landscape).
    • The Optimization Path (red line with markers) shows how m and c move together through this landscape with each step of gradient descent. You can clearly see it starts from the Starting Point and spirals down towards the Global Minima (the true m and c values where the loss is lowest).

This example perfectly illustrates how by calculating the slope (gradient) of the loss function with respect to each parameter (weight and bias) and then taking small steps in the opposite direction of that slope, we can iteratively adjust these parameters to find the set that minimizes the error and reaches the global minimum for convex problems.

import numpy as np
import matplotlib.pyplot as plt

# --- 1. Generate Synthetic Data ---
# Let's create some data that follows y = 2*x + 5, with some noise
np.random.seed(42) # for reproducibility
X = np.linspace(0, 10, 50).reshape(-1, 1) # 50 input values from 0 to 10
true_m = 2.0 # True slope (weight)
true_c = 5.0 # True intercept (bias)
y_true = true_m * X + true_c + np.random.randn(50, 1) * 2 # Add some noise

print(f"True Weight (m): {true_m}")
print(f"True Bias (c): {true_c}")
print(f"Dataset size: {X.shape[0]} samples")

# --- 2. Define Loss Function (Mean Squared Error) ---
def mse_loss(y_pred, y_true):
    """Calculates Mean Squared Error."""
    return np.mean((y_pred - y_true)**2)

# --- 3. Simple Linear Model (Our single neuron) ---
def linear_model(X, m, c):
    """Calculates prediction for a linear model."""
    return m * X + c

# --- 4. Gradient Descent for Weights (m) and Biases (c) ---
def train_linear_model(X, y_true, initial_m, initial_c, learning_rate, epochs):
    """
    Trains a simple linear model using gradient descent.
    """
    m = initial_m
    c = initial_c
    loss_history = []
    m_history = []
    c_history = []

    print(f"\n--- Training Process (Learning Rate: {learning_rate}) ---")
    for epoch in range(epochs):
        # --- Forward Pass ---
        y_pred = linear_model(X, m, c)
        loss = mse_loss(y_pred, y_true)

        # Store history for plotting
        loss_history.append(loss)
        m_history.append(m)
        c_history.append(c)

        # --- Backward Pass (Calculate Gradients) ---
        # Derivative of MSE Loss w.r.t. y_pred: dLoss/dy_pred = 2 * (y_pred - y_true) / N
        # (The 2 and N cancel out with the sum for practical gradient descent,
        # so we often just use (y_pred - y_true) as the core error term)
        error_term = (y_pred - y_true) # This is the core error signal (dLoss/dy_pred)

        # Gradient for weight (m): dLoss/dm = (dLoss/dy_pred) * (dy_pred/dm)
        # dy_pred/dm = X (from y_pred = m*X + c)
        gradient_m = np.mean(error_term * X) # Average gradient over all samples

        # Gradient for bias (c): dLoss/dc = (dLoss/dy_pred) * (dy_pred/dc)
        # dy_pred/dc = 1 (from y_pred = m*X + c)
        gradient_c = np.mean(error_term * 1) # Average gradient over all samples

        # --- Update Weights and Bias ---
        m = m - learning_rate * gradient_m
        c = c - learning_rate * gradient_c

        if epoch % (epochs // 10) == 0: # Print every 10% of epochs
            print(f"Epoch {epoch:4d} | Loss: {loss:.4f} | Current m: {m:.4f} | Current c: {c:.4f}")
            if epoch == 0:
                 print(f"    Initial Gradients: dm={gradient_m:.4f}, dc={gradient_c:.4f}")

    print(f"Epoch {epochs-1:4d} | Loss: {loss:.4f} | Final m: {m:.4f} | Final c: {c:.4f}")
    return m, c, loss_history, m_history, c_history

# --- 7. Visualization of Weight and Bias Trajectory (Loss Landscape) ---
# This is a conceptual visualization of the "loss landscape"

# Create a grid of possible m and c values
m_values = np.linspace(0, 4, 50)
c_values = np.linspace(0, 10, 50)
M, C = np.meshgrid(m_values, c_values)

# Calculate loss for each combination of m and c
Z_loss = np.array([mse_loss(linear_model(X, m, c), y_true) for m, c in zip(np.ravel(M), np.ravel(C))])
Z_loss = Z_loss.reshape(M.shape)

fig = plt.figure(figsize=(10, 8))
ax = fig.add_subplot(111, projection='3d')

# Plot the loss surface
ax.plot_surface(M, C, Z_loss, cmap='viridis', alpha=0.6)

# Plot the trajectory of m and c during training
ax.plot(m_hist, c_hist, loss_hist, color='red', linewidth=3, marker='o', markersize=5, label='Optimization Path')

ax.scatter(true_m, true_c, mse_loss(linear_model(X, true_m, true_c), y_true),
           color='blue', s=100, marker='*', label='Global Minima (True Values)')
ax.scatter(initial_m, initial_c, mse_loss(linear_model(X, initial_m, initial_c), y_true),
           color='cyan', s=100, marker='^', label='Starting Point')

ax.set_xlabel('Weight (m)')
ax.set_ylabel('Bias (c)')
ax.set_zlabel('Loss (MSE)')
ax.set_title('Loss Landscape with Gradient Descent Path')
ax.legend()
plt.show()
